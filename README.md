# 古籍校对

#### 介绍
一个古籍校对工具，主要用来校对古籍，当然不是古籍也可以。除了古籍校对外还有繁体字简体字互相转换，繁体字异体互相转换，书籍比对，书籍里相似文本搜索等。

#### 开发环境
Vue3+Electron+lowdb+express+element plus

#### 详细使用说明
https://www.yuque.com/woniu-avesf/tvyxhx/sg007f
